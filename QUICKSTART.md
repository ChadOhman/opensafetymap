## 1. Clone the repo
```bash
git clone https://github.com/ChadOhman/opensafetymap.git
cd opensafetymap
```

## 2. Import the database
- Create a MySQL database (e.g., `accidents`).  
- Import the schema + seed data:
```bash
mysql -u your_user -p accidents < seed.sql
```

